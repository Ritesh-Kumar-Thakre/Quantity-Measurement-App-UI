# 📦 Quantity Measurement App (Frontend)

This repository contains frontend implementations of the **Quantity Measurement Application**, built using two different approaches across separate branches.

---

## 🌿 Branch Structure

| Branch Name                        | Description                                 |
| ---------------------------------- | ------------------------------------------- |
| `feature/UC1-frontend-html-css-js` | Frontend using HTML, CSS, JavaScript & AJAX |
| `feature/UC2-frontend-angular`     | Frontend using Angular                      |

---

## 🚀 Use Case 1 (UC1): HTML, CSS, JS & AJAX

### 🔧 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* AJAX

### 💡 Key Concepts Covered

* DOM Manipulation
* Event Handling
* Forms & Validation
* Objects & Classes
* Exception Handling
* Async Programming (Callbacks, Promises)
* AJAX API Calls
* Dynamic UI Rendering
* Conditional Logic

### 🎨 CSS Topics

* Float & Layout Techniques
* Flexbox & Grid Layout
* Complex Selectors
* Pseudo Classes & Elements
* Semantic HTML Structure
* Responsive Design (Media Queries)
* Typography & Styling
