# 📏 Quantity Measurement Web App

![App Preview](frontIMAGE.png)

---

## 🚀 Overview
The **Quantity Measurement Web App** is a modern web application that allows users to perform various operations on physical quantities such as:

- 📏 Length  
- ⚖️ Weight  
- 🌡️ Temperature  
- 🥤 Volume  

Users can **convert, compare, and perform arithmetic operations** on different measurement units seamlessly.

This project is built using:
- 🎨 Frontend: HTML, CSS, JavaScript  
- ⚙️ Backend: Spring Boot (REST API + JWT Authentication)

---

## ✨ Features

### 🔐 Authentication
- User Signup & Login
- Secure JWT-based authentication
- Token stored in browser (localStorage)

---

### 📊 Measurement Operations

#### 1. Conversion 🔁
Convert between units:
- Feet → Inches  
- Celsius → Fahrenheit  
- Litres → Gallons  

#### 2. Comparison ⚖️
Compare two quantities:
- Equal / Greater / Smaller

#### 3. Arithmetic ➕
Perform operations:
- Addition (+)
- Subtraction (-)
- Division (/)

---

### 🎯 Supported Measurement Types

| Type         | Units Available |
|--------------|----------------|
| Length       | FEET, INCHES, YARD, CENTIMETRE |
| Weight       | KG, GRAM, POUND |
| Temperature  | CELSIUS, FAHRENHEIT, KELVIN |
| Volume       | MILILITRE, LITRE, GALLON |
